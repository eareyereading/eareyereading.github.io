我可以取消試用或訂閱嗎？

可以。開通會員後，在選擇取消之前，會員會自動續訂。您可按照以下步驟取消會員：

  1. 打開 iPhone/iPad 上的「設定」應用。
  2. 輕點螢幕頂部的「您的姓名」。
  3. 輕點「訂閱」。
  4. 輕點您想要管理的訂閱。
  5. 輕點「取消訂閱」。

取消後，您的會員將在當前帳單週期結束時停止。

若當前您正在使用蘋果手機查看，點擊此處即可進入蘋果訂閱取消頁面。

如何更改付費方案？

您可以通過設備設定更改訂閱方案：

  1. 打開設備上的「設定」應用。
  2. 輕點您的姓名，然後輕點「訂閱」。
  3. 輕點您想要更改的訂閱。
  4. 輕點「查看所有方案」，然後按照螢幕上的說明完成購買。

關於可用方案的詳細資訊，請查看會員方案頁面。

如何查看訂閱狀態？

查看訂閱詳情的方法：

  1. 打開設備上的「設定」應用。
  2. 輕點螢幕頂部的「您的姓名」。
  3. 輕點「訂閱」。
  4. 輕點訂閱，即可查看方案詳情、續訂日期或到期日期。

如何獲取發票？

發票由 App Store 發送到與您的 Apple ID 關聯的電子郵箱。如果您沒有收到發票，請直接聯繫蘋果客服。因為我們通過 App Store 銷售訂閱，所以我們無法自行開具發票。

如何申請退款？

如果您希望退款，需要直接聯繫蘋果：

  1. 訪問 https://reportaproblem.apple.com
  2. 使用您的 Apple ID 登入
  3. 找到您的購買記錄並提交退款請求

如有其他問題，請聯繫我們：eareyereading@gmail.com
