Release Notes

8.9.76

Release Date: 2025.07.23

  • Support for embedded full-text translation, which can display translations directly below paragraphs.
  • Optimized PDF OCR processing, improving OCR recognition accuracy and performance for PDF documents.

8.9.75

Release Date: 2025.07.15

  • Fixed crashes and improved stability

8.9.74

Release Date: 2025.07.09

  • Support for split-screen full-text translation, enhancing reading experience
  • Support for quickly adding polysemous words
  • Lifetime membership supports upgrade discounts
  • Optimized iCloud sync logic, improving data synchronization stability
  • Fixed the issue where user data was accidentally deleted when iCloud had no available space

8.9.73

Release Date: 2025.06.27

  • Improved app performance and reliability

8.9.72

Release Date: 2025.06.25

  • Optimized iCloud sync
  • Optimized built-in translation engine
  • Fixed the issue where system dictionary was unavailable

8.9.71

Release Date: 2025.06.21

  • Removed in-app ads.

8.9.70

Release Date: 2025.06.20

  • Fixed crash issues

8.9.69

Release Date: 2025.06.18

  • Fixed the issue of repeated update prompts
  • Fixed the issue where popup errors incorrectly indicated reading position changes
  • When push notifications are enabled, supports real-time synchronization of multi-device data via iCloud
  • Fixed the issue where pages didn’t automatically turn when looping cross-page sentences

8.9.68

Release Date: 2025.06.16

  • Fixed crash issues

8.9.67

Release Date: 2025.06.11

  • Utilizes iCloud to automatically sync data, maintaining consistency across multiple devices

8.9.66

Release Date: 2025.04.10

  • Fixed the issue where the first page of books couldn’t be read aloud
  • Keep screen highlighted during study
  • Fixed crashes and improved stability

8.9.65

Release Date: 2025.03.29

  • Added a blur intensity setting to the “Blur Text” feature, allowing adjustment of the blur effect.
  • Optimized subscription experience, restoring the purchase of Reader Membership and Unlimited Reading Membership.
  • Improved vocabulary analysis, resolving the issue where adding to the study list would easily open details.
  • When viewing study details in the study list, you can now switch directly to the previous/next study on the details page.
  • Fixed the issue where sentence playback intervals were ineffective during TTS reading.
  • Restored the split-screen experience on the iPad/Mac study page.
  • Added support for more keyboard shortcuts.
  • Fixed the issue where notes could not be added when the keyboard was minimized on iPad.
  • Fixed the issue where, during book reading, chapters could not switch automatically if the audio for the next chapter was not downloaded.
  • Fixed the issue where clicking on a word in a PDF on Mac could not trigger word lookup.

8.9.64

Release Date: 2025.02.25

  • Optimized the feature to export study content in Anki format, supporting the export of images in study content to Anki.
  • Improved the display effect of PDF text highlights in night mode.
  • When Blur Text is enabled, you can toggle the blur state by clicking on the text.
  • Added the ability to translate sentences via the Eudic App.
  • Fixed known crash issues.
