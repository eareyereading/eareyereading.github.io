title: 導入書籍

可以通過多種方式導入書籍到聽閱中閱讀。當前支持的書籍格式包括：EPUB、MOBI、PDF、RTF和TXT等。

如果您有其他格式的書籍，您可下載免費軟件 Calibre 將書籍轉換為epub格式，然後導入聽閱。Calibre支持以下格式書籍：AZW，AZW3，AZW4，CBZ，CBR，CBC，CHM，DJVU，DOCX，EPUB，FB2，FBZ，HTML，HTMLZ，LIT，LRF，MOBI，ODT，PDF，PRC，PDB，PML，RB，RTF，SNB，TCR，TXT，TXTZ。

WIFI 導入

  1. 將手機和電腦連上同一個WIFI網絡。
  2. 在聽閱首頁點按左上角“+”。
  3. 選擇“WIFI導入”打開WIFI傳書頁面。
  4. 在電腦上打開瀏覽器，輸入傳書頁面顯示的地址並打開，然後將書籍拖入網頁指定位置並上傳。
  5. 在手機端關閉傳書頁面，聽閱將導入書籍並顯示在書庫頁面。

Files 導入

  1. 將手機通過USB連接電腦。
  2. 通過iTunes把書籍導入聽閱目錄。
  3. 在手機上打開Files應用，點按“我的iPhone”->“聽閱”。
  4. 點擊書籍檔案將打開聽閱並導入書籍。

微信導入

  1. 將書籍發送給微信的“檔案傳輸助手”。
  2. 輕點書籍。
  3. 輕點“用其他應用打開”。
  4. 選擇「拷貝到“聽閱”」，即可將書籍導入到聽閱閱讀。

百度網盤導入

  1. 在iPhone/iPad中打開“百度網盤”。
  2. 下載您想閱讀的書籍。
  3. 等下載完成後，輕點書籍頁面左下角的“打開”。
  4. 選擇「拷貝到“聽閱”」，即可將書籍導入到聽閱閱讀。
