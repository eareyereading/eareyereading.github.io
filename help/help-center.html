我可以取消试用或订阅吗？

可以。开通会员后，在选择取消之前，会员会自动续订。您可按照以下步骤取消会员：

  1. 打开 iPhone/iPad 上的”设置”应用。
  2. 轻点屏幕顶部的”您的姓名”。
  3. 轻点”订阅”。
  4. 轻点您想要管理的订阅。
  5. 轻点”取消订阅”。

取消后，您的会员将在当前账单周期结束时停止。

若当前您正在使用苹果手机查看，点击此处即可进入苹果订阅取消页面。

如何更改付费方案？

您可以通过设备设置更改订阅方案：

  1. 打开设备上的”设置”应用。
  2. 轻点您的姓名，然后轻点”订阅”。
  3. 轻点您想要更改的订阅。
  4. 轻点”查看所有方案”，然后按照屏幕上的说明完成购买。

关于可用方案的详细信息，请查看会员方案页面。

如何查看订阅状态？

查看订阅详情的方法：

  1. 打开设备上的”设置”应用。
  2. 轻点屏幕顶部的”您的姓名”。
  3. 轻点”订阅”。
  4. 轻点订阅，即可查看方案详情、续订日期或到期日期。

如何获取发票？

发票由 App Store 发送到与您的 Apple ID 关联的电子邮箱。如果您没有收到发票，请直接联系苹果客服。因为我们通过 App Store 销售订阅，所以我们无法自行开具发票。

如何申请退款？

如果您希望退款，需要直接联系苹果：

  1. 访问 https://reportaproblem.apple.com
  2. 使用您的 Apple ID 登录
  3. 找到您的购买记录并提交退款请求
