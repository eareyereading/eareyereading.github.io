导入书籍

听阅支持多种方式导入书籍，方便您随时随地阅读。当前支持的书籍格式包括 EPUB、MOBI、PDF、RTF 和 TXT 等。

如果您的书籍格式不受支持，可以使用免费软件 Calibre 将其转换为 EPUB 格式后导入听阅。Calibre 支持转换以下格式：AZW、AZW3、AZW4、CBZ、CBR、CBC、CHM、DJVU、DOCX、EPUB、FB2、FBZ、HTML、HTMLZ、LIT、LRF、MOBI、ODT、PDF、PRC、PDB、PML、RB、RTF、SNB、TCR、TXT、TXTZ。

WIFI 导入（适用于无线传输）

  1. 确保 手机和电脑连接到同一 Wi-Fi 网络。
  2. 在听阅首页，点按左上角 图标。
  3. 选择 “WIFI 导入”，进入 Wi-Fi 传书页面。
  4. 在电脑浏览器中输入 传书页面显示的地址 并打开。
  5. 拖拽书籍文件 至网页指定区域进行上传。
  6. 传输完成后，关闭手机端传书页面，书籍将自动导入并显示在书库中。

Files 导入（适用于 USB 连接）

  1. 使用 USB 连接手机与电脑。
  2. 通过 iTunes 将书籍文件添加到 听阅目录。
  3. 在手机上打开 “文件” 应用，进入 “我的 iPhone” → “听阅”。
  4. 轻点书籍文件，即可自动导入听阅阅读。

微信导入（适用于微信文件）

  1. 将书籍文件发送至 微信 “文件传输助手”。
  2. 轻点收到的书籍文件。
  3. 选择 “用其他应用打开”。
  4. 选择 “拷贝到听阅”，书籍即导入听阅阅读。

百度网盘导入（适用于网盘存储）

  1. 在 iPhone/iPad 上打开 “百度网盘”。
  2. 下载您想阅读的书籍。
  3. 下载完成后，轻点书籍，选择 左下角“打开”。
  4. 选择 “拷贝到听阅”，书籍即导入听阅阅读。
